thundreX

A neumorphic design system and web component library. Soft UI with clean aesthetics, full theming, Tailwind CSS integration, and WCAG AA accessibility.

$ npm install @thundrex/web-components

Theming

Override any design token via CSS custom properties, or use the built-in setTheme() helper for instant runtime switches.

JavaScript
import { setTheme, resetTheme } from '@thundrex/web-components';

// Built-in presets
setTheme('dark');
setTheme('ocean');
resetTheme();

// Custom tokens
setTheme({
  accent: '#E040FB',
  primary: '#00BCD4',
  bg: '#EAEAEA',
  neuDark: 'rgba(180,180,180,0.2)',
  neuLight: 'rgba(255,255,255,0.7)',
});

// Scope to a subtree
setTheme('ocean', document.querySelector('.sidebar'));

Tailwind CSS Integration

Use thundrex design tokens as Tailwind utility classes. All values reference CSS custom properties, so they respond to runtime theme switches.

Setup

tailwind.config.js
import thundrex from '@thundrex/web-components/tailwind-preset';

export default {
  presets: [thundrex],
  content: ['./src/**/*.{html,js,ts,jsx,tsx}'],
};

Usage Example

HTML
<!-- Neumorphic card using Tailwind classes -->
<div class="bg-tx-bg rounded-tx-lg shadow-tx-md p-tx-6 font-tx">
  <h2 class="text-tx-text text-tx-xl font-bold">Card Title</h2>
  <p class="text-tx-text-secondary text-tx-sm mt-tx-2">Body text</p>
  <input class="bg-tx-bg rounded-tx shadow-tx-inset-sm p-tx-3 mt-tx-4" />
</div>
