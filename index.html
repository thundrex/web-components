ThunderX

A lightweight Neumorphic (Soft UI) component library built with Lit web components. Drop them into any framework or vanilla HTML.

$ npm install thunderx

Button

Extruded by default, lifts on hover, and presses inward on click. Supports multiple sizes, pill shape, icon-only, and disabled state.

Variants

Default Small Large Pill Disabled
HTML
<neu-button>Default</neu-button>
<neu-button size="sm">Small</neu-button>
<neu-button size="lg">Large</neu-button>
<neu-button pill>Pill</neu-button>
<neu-button disabled>Disabled</neu-button>
<neu-button icon-only>...</neu-button>

API

PropertyTypeDefaultDescription
size'sm' | 'md' | 'lg''md'Button size variant
pillbooleanfalseFully rounded corners
icon-onlybooleanfalseCircular button for icons
disabledbooleanfalseDisables interaction

Input

Form inputs default to the recessed (inset) neumorphic look. Includes text fields, email, and a search variant with a magnifying-glass icon.

Text Fields

HTML
<neu-input label="Full Name" placeholder="John Doe"></neu-input>
<neu-input label="Email" type="email"></neu-input>

Search Bar

HTML
<neu-input search placeholder="Search anything..."></neu-input>

API

PropertyTypeDefaultDescription
labelstring''Label above the input
namestring''Form field name
placeholderstring''Placeholder text
valuestring''Current value
typestring'text'HTML input type
searchbooleanfalseSearch mode with icon
EventDetailDescription
neu-input{ value: string }Fires on input change

Toggle

A switch control with a recessed track and an extruded draggable thumb. The active state lights up with the accent color.

States

HTML
<neu-toggle label="Wi-Fi"></neu-toggle>
<neu-toggle label="Bluetooth" checked></neu-toggle>

API

PropertyTypeDefaultDescription
checkedbooleanfalseToggle state
labelstring''Adjacent label text
EventDetailDescription
neu-change{ checked: boolean }Fires on toggle

Checkbox

Extruded when unchecked, recesses inward with an accent checkmark when checked. Fully keyboard accessible.

States

HTML
<neu-checkbox label="Accept terms"></neu-checkbox>
<neu-checkbox label="Newsletter" checked></neu-checkbox>

API

PropertyTypeDefaultDescription
checkedbooleanfalseChecked state
labelstring''Clickable label text
EventDetailDescription
neu-change{ checked: boolean }Fires on check/uncheck

Slider

A range input with a recessed track and an extruded thumb bordered in the accent color. Supports labels and live value display.

Variants

HTML
<neu-slider
  label="Volume"
  show-value
  value="65"
></neu-slider>

<neu-slider
  label="Temperature"
  show-value
  value="22"
  min="16"
  max="30"
  step="0.5"
></neu-slider>

API

PropertyTypeDefaultDescription
valuenumber50Current slider value
minnumber0Minimum value
maxnumber100Maximum value
stepnumber1Step increment
labelstring''Label text
show-valuebooleanfalseShow live value
EventDetailDescription
neu-input{ value: number }Fires on value change

Card

A content container with header, body, and footer slots. Supports extruded (default), flat (recessed), and interactive modes.

Variants

Extruded Card

Standard raised neumorphic card. Content sits naturally on the surface.

Action

Recessed Card

Uses the flat attribute for an inset well.

Learn More

Interactive Card

Hover and press to see shadow transitions. Great for clickable items.

Nested Controls
HTML
<neu-card>
  <h3 slot="header">Title</h3>
  <p>Card body content goes here.</p>
  <div slot="footer">
    <neu-button size="sm">Action</neu-button>
  </div>
</neu-card>

<!-- Recessed variant -->
<neu-card flat>...</neu-card>

<!-- Interactive (hover + press) -->
<neu-card interactive>...</neu-card>

API

PropertyTypeDefaultDescription
flatbooleanfalseRecessed / inset look
interactivebooleanfalseHover and press effects
SlotDescription
headerCard title area
(default)Main card body
footerFooter with actions
